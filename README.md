# agr-forte-futuro-sustentavel
Um projeto para o Agrinho 2026 que nasceu da ideia de que produção agrícola e meio ambiente não precisam ser inimigos. No Paraná e no Brasil, cada vez mais o agro mostra que é possível plantar, colher e ainda cuidar da terra: são dois lados do mesmo ser, que crescem juntos quando há equilíbrio.
